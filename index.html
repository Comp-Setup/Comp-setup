Як користуватися завантажувальною флешкою?

  1. Вставте флешку у вимкнений ПК або ноутбук.
  2. При увімкненні натискайте клавішу виклику Boot Menu (зазвичай F12, F11, F8 або Esc).
  3. Оберіть вашу флешку у списку та натисніть Enter.
  4. У меню оберіть потрібну ОС або пункт Live CD для порятунку файлів.
